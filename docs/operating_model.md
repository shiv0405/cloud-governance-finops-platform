# Operating Model

Cloud Governance and FinOps Intelligence is built to support a monthly or weekly cloud posture review for platform, security, and FinOps teams.

## Core Outputs

- `account_scorecard.csv`
  - ranked account-level view of risk, spend, governance, and optimization posture
- `portfolio_kpis.json`
  - headline metrics for leadership review
- `remediation_recommendations.csv`
  - owner-aligned actions for the highest-priority accounts
- `executive_brief.md`
  - short-form narrative for steering reviews

## Team Usage Pattern

1. Run data refresh and reporting generation
2. Review top-risk accounts and savings opportunities
3. Assign actions across cloud platform, security engineering, and FinOps
4. Track whether governance posture improves month over month
